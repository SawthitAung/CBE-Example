# ⚓️ Marine Study Hub (CBE System)

**Marine Study Hub** သည် Marine Engineer များအတွက် **Certificate of Competency (COC)** စာမေးပွဲများတွင် ကြုံတွေ့ရမည့် **Computer Based Examination (CBE)** စနစ်ကို အထောက်အကူပြုစေရန် ဖန်တီးထားသော လေ့လာရေး Website တစ်ခု ဖြစ်သည်။
---

## 🛠 နည်းပညာများ (Tech Stack)
*   **Frontend:** HTML5, CSS3 (Custom UI), JavaScript (ES6 Modules)
*   **Backend:** Firebase Realtime Database
*   **Hosting:** GitHub Pages
*   **Icons:** Font Awesome 6.4.0
